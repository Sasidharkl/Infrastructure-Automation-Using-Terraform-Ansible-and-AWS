# Infrastructure Automation using Terraform, Ansible, and AWS

## Project Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform and configuration management using Ansible to automate AWS infrastructure provisioning and server setup.

## Technologies Used

- Terraform
- Ansible
- AWS (EC2, VPC, RDS)
- GitHub Actions (CI/CD)

## Architecture

Terraform provisions:

- VPC
- Public Subnet
- EC2 Instance
- RDS MySQL Database

Ansible configures:

- Nginx Web Server
- Application deployment

CI/CD pipeline automatically deploys infrastructure using GitHub Actions.
